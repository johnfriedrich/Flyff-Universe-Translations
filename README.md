## How to Contribute üõ†Ô∏è

Contributions are welcome and appreciated. Please follow the rules and steps below carefully.

### Contribution Rules (Important)

- **All files must be encoded in UTF-8.**
- **Do not add, delete, or change the order of translation entries** in any file.
- You may:
  - Translate existing English text into your target language, **and**
  - **Correct or improve existing translations** if there is a mistake, inconsistency, or quality issue.
- Do not modify keys, identifiers, formatting, or file structure.
- Ensure translations are accurate, consistent, and respectful of the game‚Äôs tone.

Pull requests that do not follow these rules may be rejected.

### Contribution Steps

1. **Fork** this repository.
2. Create a new branch for your changes.
3. Edit the relevant translation files following the rules above.
4. Commit your changes with a clear message.
5. Submit a **Pull Request** with a brief description of your changes.

Our team will review your contribution and may request adjustments before merging.

## Repository Scope & Conduct üö´

This repository is **strictly limited to translation and localization work** for Flyff Universe.

Please **do not open issues or pull requests** for any of the following purposes:
- Customer support requests
- Game bug reports
- Gameplay issues or balance feedback
- Complaints or moderation requests
- Feature suggestions or general discussions
- Any topic not directly related to translation content

Issues or pull requests that fall outside the scope of translations will be **closed without review**.

Repeated misuse of issues or pull requests, or failure to follow these rules, may result in the contributor being **restricted or banned from this repository**.

For game-related issues or support, please use the **official Flyff Universe support channels**.
